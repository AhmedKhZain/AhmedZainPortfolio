Samraa AlYaman

2026 | E-commerce Platform API
▼

Samraa AlYaman is a high-performance, enterprise-grade E-commerce platform designed around distributed system principles. The primary engineering goals of the platform are absolute data consistency, high-throughput query performance, reliable asynchronous workflows, and complex business domain modeling.

Project Core & Engineering Overview

  • Distributed Event Sourcing: Designed a complete historical state tracking workflow for full auditability of every business operation.
  • Eventual Consistency Middleware: Fault-tolerant pipeline that publishes domain events after the HTTP response completes, with automatic rollback on failure.
  • Hybrid Persistence Layer: EF Core for transactional writes and Dapper for high-performance read queries.
  • Selective Transaction Strategy: MediatR pipeline behaviors use marker interfaces to start transactions only when event synchronization is needed.
  • Soft Foreign Key Cascade: Simulates cascading deletes through domain events instead of database-level cascade, preventing relational lockups.
  • Advanced Audit Subsystem: Entities generate domain events while simultaneously creating immutable audit records for every business state transition.

G Search

2025 | Custom Search Engine
▼

A Python-based web crawler and HTML parser that builds an inverted index from crawled pages, served via a .NET RESTful API for search queries.

Project Core & Engineering Overview

  • Python Web Crawler: Discovers and fetches web pages using requests, urllib.parse, with concurrent threading via concurrent.futures.ThreadPoolExecutor.
  • HTML Parser: Parses fetched HTML with BeautifulSoup to extract text and links, tracking visited URLs with collections.deque.
  • Inverted Index Generation: Processes parsed content into inverted index files persisted using os and shutil, with progress reported via tqdm.
  • .NET RESTful API: Serves search queries against the generated inverted index.
