Slide 2: Project Core & Security Overview (Text-Only Slide)
Title: Project Core & Security Overview

Content Bullet Points:

Patient-Centric Mission: Bridges doctors, patients, and guardians through secure medical pipelines, prescription synchronization, and real-time reminders.

Decoupled Architecture: Built using ASP.NET Core and Clean Architecture to isolate mechanical concerns from domain layers.

Patient-Guardian Security Module: Enables caretakers to oversee doctor-prescribed medication schedules and receive compliance updates.

Role-Based Access Control (RBAC): Enforces strict authorization policies over patient health information, billing execution, and system pathways.

Secure Payment Infrastructure: Integrates full Stripe Gateway operations utilizing robust transactional consistency rules.
